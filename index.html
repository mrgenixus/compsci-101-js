Code Dojo Loveland

Welcome to the Dojo

Code Dojo Loveland is a virtual space for creators to find their secret heart as they explore skills and fields which have the potential to to become true passions.

We hope to take every student on a journey of self-discovery: that they will find undiscovered potential to imagine and reason about computers and technology.

Our goal is to give each one the experiences and disciplines that will build confidence and allow their creative spirit to flourish.

Fundamentals of Computer Programming

Learn introductory computer science in this 7-month workshop on web application development. Classes are held on the first and third Fridays of each month at the Loveland CreatorSpace. Our first session is January 2nd at 6 p.m. At the end of each month, the CreatorSpace will host a public showcase for students to present their work.

This workshop series is based on the JavaScript language. Upon completing the curriculum, students will understand how to build complete web applications using various JavaScript libraries and frameworks.

There is no fee for the course, however, donations are accepted.
